# COEPD AI Advisor

COEPD AI Advisor is an AI-powered chatbot and lead generation platform designed to assist prospective students with Business Analyst training information.

Features
- AI chatbot for student queries
- Lead capture system
- Admin dashboard
- Business Analyst course information
- Automated responses

Tech Stack
- FastAPI
- Python
- HTML / CSS / JavaScript
- Jinja Templates

Deployment
Hosted using cloud platforms like Render or Railway.

Render Start Command
uvicorn main:app --host 0.0.0.0 --port $PORT --workers 2 --proxy-headers
